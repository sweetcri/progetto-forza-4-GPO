# progetto-forza-4-GPO
 
Lo scopo del progetto è lo sviluppo di un software interattivo del gioco Forza 4, che permetta a due giocatori di sfidarsi in locale su un’interfaccia grafica intuitiva. Il programma dovrà gestire automaticamente le regole del gioco, determinare il vincitore e consentire una nuova partita in modo immediato.
